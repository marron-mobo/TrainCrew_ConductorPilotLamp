**「車掌用知らせ灯+発車放送」プラグイン**

***「車掌用知らせ灯」機能***
カーブのある駅などで車側灯が前まで見えないときに、京阪電車のような車掌用知らせ灯があると便利だと思い開発しました。（実際には結構見にくそうな位置にあるので車掌用知らせ灯の確認が大変そうです。）

JR等だとドラムスイッチ「後」位置のとき知らせ灯は点灯しませんが、京阪電車ではドラムスイッチの位置にかかわらず（すなわち制御スイッチの入切にかかわらず）点灯します。

***「発車放送機能」***

主にワンマン列車で、戸閉に連動して車外スピーカから放送を鳴らす機能を模したプラグインです。

departure.wavをお好きな音声に変えることができます。(デフォルト:VOICEVOX 四国めたん)

exeファイル下部に「発車放送が見つかりました」と表示されたときに再生されます。再生機能が不要な場合はdeparture.wavを削除してください。
