**「車掌用知らせ灯」プラグイン**

カーブのある駅などで車側灯が前まで見えないときに、京阪電車のような車掌用知らせ灯があると便利だと思い開発しました。（実際には結構見にくそうな位置にあるので車掌用知らせ灯の確認が大変そうです。）
JR等だとドラムスイッチ「後」位置のとき知らせ灯は点灯しませんが、京阪電車ではドラムスイッチの位置にかかわらず（すなわち制御スイッチの入切にかかわらず）点灯します。
